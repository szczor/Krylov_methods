This project contains implementations of algorithms used to solve huge linear systems called Krylov methods. They come from the book
'Iterative Methods for Sparse Linear Systems' by Yousef Saad. In particular we implement: GMRES with housholder and gram-schmidt orthogonalisation, ORTHOMIN,
ORTHODIR and GCR. As a test example, we have implemented a differential equation, which is transformed into a linear system by the Finite Difference method.
