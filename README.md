This project contains implementations of alghoritms used to solve huge linear systems called Krylov methods from the book
'Iterative Methods for Sparse Linear Systems' by Yousef Saad. In particular: GMRES with housholder and gram-schmidt orthogonalisation, ORTHOMIN,
ORTHODIT and GCR. As the test example, we have a differential equation, which is transformed into a linear system by the Finite Difference method.
